# **App Name**: 내집 코디

## Core Features:

- 간편 로그인: 카카오/네이버 OAuth를 통한 보안 로그인 및 게스트 모드로 즉시 서비스 체험.
- 직장 주소 입력 및 모드 선택: 두 직장 주소를 입력하고 '커플' 또는 '싱글' 진단 모드를 선택합니다.
- 교통편 고려 동선 교집합 진단: 두 직장 주소 기반의 출퇴근 동선을 분석하여 가장 합리적인 거주지 교집합 지역을 계산합니다.
- 스마트 동네 분석 툴: 교통 데이터와 도시 데이터를 분석하여 통근 균형 및 사용자 선호도에 따른 최적 동네를 제안하는 AI 기반 툴.
- 진단 결과 지도 시각화: 카카오맵 위에 진단된 후보 동네와 출퇴근 경로 정보를 명확하게 시각화하여 표시합니다.
- 상세 교통 정보 제공: 선택된 후보 동네별 통근 시간, 교통수단, 환승 정보를 한눈에 볼 수 있도록 제공합니다.
- 결과 공유 링크: 진단 결과를 고유한 URL을 통해 배우자나 친구와 카카오톡 등으로 공유할 수 있습니다.

## Style Guidelines:

- 주요 색상: 신뢰와 현대적인 느낌을 주는 진한 파랑색 (#3985C0).
- 배경 색상: 깔끔하고 차분한 분위기를 연출하는 아주 밝은 회청색 (#F4F7F8).
- 포인트 색상: 상쾌하고 활기찬 느낌을 더하는 청록색 (#63DBC6).
- Kakao 버튼 색상: #FEE500 (카카오), Naver 버튼 색상: #03C75A (네이버)를 각 브랜드 가이드라인에 따라 사용합니다.
- 모든 텍스트에 높은 가독성의 한글 폰트인 'Pretendard' (Sans-serif)를 적용합니다.
- 지오로케이션, 경로, 집, 사람 등 주요 기능을 나타내는 명확하고 현대적인 라인 아이콘을 사용합니다.
- 모바일 우선(최소 375px 기준) 반응형 디자인으로, 태블릿(768px) 이상에서는 좌우 분할 또는 사이드 패널 레이아웃을 제공합니다. 최소 터치 영역은 44px로 유지합니다.
- 지도 상호작용 및 패널 전환 시 부드러운 전환 효과를 적용하여 사용자 경험을 향상시킵니다.
